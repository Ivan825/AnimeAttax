# Anime Attax

## Prerequisites

- **Java JDK 17** installed on your system. However Java JDK 23 is preferred
- **Maven** not required to be installed on your system
- **JavaFX** not required to be installed on your system.

## Project Setup

After cloning the repository, run the following command in the project directory to set up the project and download all dependencies:
## Run the following:
````
git clone https://github.com/Ivan825/AnimeAttax.git
````
CD to root directory

### On Unix/Mac:

```bash
./mvnw clean install

```
### On Windows:

````bash
mvnw.cmd clean install
````
